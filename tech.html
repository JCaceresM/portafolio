Git - Github

En la actualida un desarrollador de software debe conocer algun sistema de control de versiones para un continuo desarrollo eficiente, dinamico, controlado. Unas de las cosas que siempre me han recomendado y que yo he recomendado es aprender git, tiene la pecurialidad de adaptarse a tu workflow muy facilmente debido a sus bastos comandos y funcionalidades. Github es el complemento ideal para git ya que este guarda en la nuve todo tu trabajo sin temor a perder tu progreso, compartir tu codigo , hostear tus paginas web, pipelines, entre muchas otras cosas geniales.
